## Установка проекта

0.	Убедитесь, что в системе установлены nodejs-v25.1 и выше, php-v8.5 и выше и composer
1.	Скачайте репозиторий
2.	Выполните composer install
3.	Выполните npm install
4.	Выполните php artisan key:generate
5.	Выполните php artisan migration
6.	Выполните php artisan db:seed
7.	Выполните php artisan storage:link
8. 	Для сборки используйте npm run build, npm run dev или npm run watch.
9.	Запустите сервер через php artisan serve
